Overview

Platform Overview

PPCPOKP provides a user-friendly web interface that contains five modules: Browse, Search, Tool, Submit and Download. Together, these modules support model exploration, retrieval, guided recommendation, structured comparison, data contribution, and resource reuse.

Browse

Explore model records by study type and inspect structured study details.

Search

Perform precise structured retrieval or search multiple records in one submission.

Tool

Use model recommendation and model comparison features for decision support.

Submit & Download

Contribute new content and obtain downloadable datasets and supporting files.

Use the navigation panel on the right to jump directly to each section. On smaller screens, the navigation moves above the content for easier access.
Module

Browse

The Browse page is used to explore prediction models in the database. Models are organized by study type and presented in three sections: Model Development, Model External Validation, and Model External Comparison. Each section is displayed on a separate page.

1. Model Development

This page presents prediction model development studies for PPCs. Models are displayed in a card-based format with summarized information on model characteristics and performance. Filtering options are also provided so that models can be viewed according to study features, target outcomes, model types, and related attributes.

The Model Detail page presents detailed information for an individual prediction model. This page provides a summary of the model, together with links to related external validation and external comparison studies. Structured sections are further provided to present additional model-specific information.

Open Model Development

2. Model External Validation / Model External Comparison

The Model External Validation and Model External Comparison pages present studies evaluating existing prediction models. These pages display summarized study information and provide access to detailed validation or comparison results.

Model External Validation View studies that externally validate existing PPC prediction models. Model External Comparison Inspect studies that compare existing models under external evaluation settings.
Module

Search

1. Advanced Search

Advanced Search allows users to perform structured and precise searches. This mode is suitable when you already know the study features, model characteristics, outcomes, or other attributes you want to combine in a single query.

Open Advanced Search

2. Batch Search

Batch Search allows users to retrieve multiple prediction models in a single search. You can input a keyword list, for example a list of Model IDs to search the database. The list should be entered as one keyword per line.

Open Batch Search
Module

Tool

1. Model Recommendation Tool

The Model Recommendation tool provides a list of PPC prediction models that match user-defined criteria. It prioritizes models based on three factors: alignment with the criteria, quality informed by PROBAST + AI tool ratings, and usability, including whether a tool or digital implementation is available.This helps users efficiently identify models that are more relevant, higher in quality, and easier to apply in practice.

Open Model Recommendation

2. Model Comparison Tool

The Model Comparison tool provides an integrated comparison of prediction models for PPCs using a composite scoring system, rather than external validation. It summarizes overall study characteristics in a unified view to support comparative assessment across selected models.

Open Model Comparison
Module

Submit

The Submit page allows the submission of data or research outputs related to prediction models for PPCs. All submissions undergo a rigorous review process before being included in the database, ensuring data quality and scientific validity.

Open Submit
Module

Download

A range of PPCPOKP datasets is available for download on this page. Each dataset is identified by its name and an accompanying description of its contents.
